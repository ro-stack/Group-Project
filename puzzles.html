    Puzzle 8

    for Loops:

    When you know exatly how many times you want to loop through a block of code, you use the for loop.

    for (statement1; statement2; statement3) {

    ...code to be executed

    }

  • statement1 - is executed (one time) before the execution of the code block.
  • statement2 - defines the condition for executing the code block.
  • statement3 - is executed (every time) after the code block has been executed.

    • for (int i = 0; i < 5; i++) {

    System.out.println(i);

    }

    This for loop will print out 0,1,2,3,4 each on a new line.
